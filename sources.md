# Реестр источников

Все URL, на которые ссылаются статьи wiki, с указанием, какой вуз и какие данные подтверждает каждая ссылка, плюс дата последней сверки и статус.

## Формат

- **URL** — полная ссылка на конкретный документ (не на главную приёмки).
- **Вуз** — slug (`bmstu`, `mai`, `mirea`, `finuniversity`, `mtuci`).
- **Подтверждает** — какие именно поля карточки берутся отсюда.
- **Дата обращения** — `YYYY-MM-DD`.
- **Статус** — `актуален`, `отозван`, `404`, `замещён`.

## Записи

На 2026-05-22 каждая карточка вуза и каждая статья направления содержит собственный раздел `## Источники` со списком URL и датами обращения. Удобнее пользоваться источниками через эти разделы:

## Правила пополнения

1. Каждое число, появляющееся в любой карточке вуза, обязано появиться здесь хотя бы один раз через свой URL.
2. Если несколько чисел подтверждаются одной страницей, делается одна запись в реестре и перечисляются поля в колонке «Подтверждает» через запятую.
3. При обновлении wiki сначала меняем значение в карточке вуза, затем обновляем строку реестра (дату обращения или статус, если страница изменилась).
4. Запрещено добавлять записи с URL из чёрного списка (агрегаторы вроде postupi.online, рейтинги, форумы).
